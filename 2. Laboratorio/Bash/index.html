1. Il comando man

Il comando man viene usato per ottenere informazioni dal manuale per un determinato altro comando.

man [altro-comando]

2. Il comando echo

Il comando echo viene usato per mostrare a video una determinata stringa.

echo "guarda che roba!"

3. Il comando pwd

Il comando pwd ("print-working-directory") viene usato per mostrare la cartella di lavoro.

pwd

4. Il comando cd

Il comando cd ("change-directory") viene usato per cambiare la cartella di lavoro.

cd <directory>

5. Il comando ls

Il comando ls viene usato per ottenere la lista dei file in una determinata directory.

ls /home/saniales/mia-cartella

6. Il comando cp

Il comando cp viene usato per copiare un file da una posizione sorgente a una destinazione.

cp file.txt /home/saniales/file.txt

7. Il comando mv

Il comando mv viene usato per spostare o rinominare un file.

mv <sorgente> <destinazione>

8. Il comando mkdir

Il comando mkdir viene usato per creare directory.

mkdir <destinazione>

9. Il comando rmdir

Il comando rmdir viene usato per rimuovere directory vuote.

rmdir <destinazione>

10. Il comando rm

Il comando rm viene usato per rimuovere un file.

rm file.txt

11. Il comando sudo

Il comando sudo permette di eseguire operazioni come utente root.

sudo <comando>

12. Il comando touch

Il comando touch viene usato per creare un file vuoto.

touch "esempio.txt"

13. Il comando chmod

Il comando chmod permette di modificare i permessi di un file o di una directory.

chmod 400 file-prova.txt

14. Il comando cat

Il comando cat viene usato per leggere il contenuto di un file.

cat file.txt

15. Il comando chown

Il comando chown permette di cambiare il proprietario di un file.

chown file.txt utente

16. Il comando ps

Il comando ps mostra i processi attivi nel sistema.

ps aux

17. Il comando grep

Il comando grep permette di cercare una stringa all'interno di un'altra.

echo "Bella Ciaone" | grep "Ciaone"

I/O Redirection

La redirezione di input/output permette di indirizzare l'output di un comando in un file o passarlo come input per un altro comando. Ecco alcuni esempi di come funziona la redirezione:

1. Redirezione dell'output in un file

echo "Ciao Mondo" > output.txt

Questo comando scrive "Ciao Mondo" nel file output.txt. Se il file esiste già, il suo contenuto verrà sovrascritto.

2. Appendere l'output a un file

echo "Buongiorno" >> output.txt

Questo comando aggiunge "Buongiorno" alla fine del file output.txt, senza sovrascrivere il contenuto esistente.

3. Redirezione dell'input da un file

sort < input.txt

Questo comando legge il contenuto di input.txt e lo ordina.

4. Pipeline

ps aux | grep "bash"

Questo comando mostra tutti i processi e filtra solo quelli che contengono "bash".

18. Script Bash

Un file di script Bash è un file di testo contenente una sequenza di comandi Bash da eseguire. Per creare uno script Bash, puoi utilizzare un editor di testo e salvare il file con estensione .sh. Assicurati di rendere il file eseguibile con:

chmod +x script.sh

Per eseguire lo script, utilizza:

./script.sh

19. Variabili

In Bash, puoi definire variabili utilizzando la sintassi:

variabile="valore"

Per utilizzare il valore di una variabile, precedila con il simbolo $:

echo $variabile

20. Operatori Logici

In Bash, puoi utilizzare operatori logici per controllare il flusso del tuo script. Alcuni operatori logici comuni sono:

  • &&: Esegue il secondo comando solo se il primo ha successo.
  • ||: Esegue il secondo comando solo se il primo fallisce.

21. Operatori Matematici

Puoi eseguire operazioni matematiche in Bash utilizzando il comando expr o le doppie parentesi:

result=$((a + b))

22. Cicli

Bash supporta diversi tipi di cicli, come for, while, e until. Ecco un esempio di ciclo for:

for i in {1..5}; do
    echo "Numero: $i"
done

23. Funzioni

Le funzioni in Bash possono essere definite utilizzando la seguente sintassi:

function_name() {
    # Comandi
}

Puoi chiamare la funzione semplicemente usando il suo nome:

function_name

24. Permessi

In Bash, i permessi determinano chi può leggere, scrivere o eseguire un file. I permessi possono essere modificati usando il comando chmod.

Tipi di Permessi

  • R: permesso di lettura
  • W: permesso di scrittura
  • X: permesso di esecuzione

Struttura dei Permessi

I permessi sono rappresentati da tre categorie:

  • U: proprietario del file
  • G: gruppo del file
  • O: altri utenti

I permessi possono essere impostati con i seguenti valori numerici:

  • 4: lettura
  • 2: scrittura
  • 1: esecuzione

I permessi possono essere combinati sommando i valori. Ad esempio, per impostare lettura e scrittura, utilizzi 6 (4+2).

Esempi di utilizzo

1. Impostare i permessi di lettura e scrittura per il proprietario:

chmod 600 file.txt

2. Impostare i permessi di lettura, scrittura ed esecuzione per il proprietario e lettura per il gruppo e altri:

chmod 755 file.txt

3. Impostare i permessi di scrittura per tutti:

chmod a+w file.txt

4. Rimuovere i permessi di esecuzione per il gruppo:

chmod g-x file.txt
